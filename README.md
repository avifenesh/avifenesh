## What I maintain

- **[Valkey GLIDE](https://github.com/valkey-io/valkey-glide)** - official Valkey/Redis OSS client library with a Rust core and multi-language bindings. I work across reliability, TLS, timeout behavior, Java/JNI, Node/N-API, CI health, docs, and contributor support.
- **[Valkey](https://github.com/valkey-io/valkey)** - upstream Valkey contributions and support around naming, compatibility, and ecosystem behavior.
- **[GLIDE MQ](https://github.com/avifenesh/glide-mq)** - high-throughput Node.js queue on Valkey/Redis Streams, powered by Valkey GLIDE and Rust-native N-API bindings.
- **[ocaml-valkey](https://github.com/avifenesh/ocaml-valkey)** - modern OCaml 5 + Eio client for Valkey with RESP3, cluster routing, TLS/mTLS, IAM auth, pub/sub, batching, and OpenTelemetry.
- **[agent-sh](https://github.com/agent-sh)** - open-source tooling for AI-assisted development workflows, including config validation, delivery automation, research/review agents, and Linux/browser control.
